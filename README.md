# 숫자게임!

Up/Down 맞추기, 숫자야구, 스도쿠가 있는 웹 게임 모음입니다.

## GitHub Pages 배포

1. GitHub 저장소에 프로젝트 업로드
2. **Settings → Pages → Source**: `Deploy from a branch`
3. Branch: `main`, Folder: `/ (root)`
4. `index.html`이 루트에 있으면 사이트가 공개됩니다

## 로그인 · 기록 저장

- 회원가입/로그인 데이터는 **브라우저 localStorage**에 저장됩니다 (서버 없이 GitHub Pages에서 동작)
- 로그인한 계정마다 **프로필**(닉네임·사진)과 **최고 기록**이 따로 저장됩니다
- 다른 기기·브라우저와는 동기화되지 않습니다

## 최고 기록

| 게임 | 기준 |
|------|------|
| 횟수맞추기 | 최소 시도 횟수 (자릿수 + 난이도별) |
| 숫자야구 | 최단 시간 (자릿수별) |
| 스도쿠 | 최단 시간 (난이도별) |

## 로컬 실행

`index.html`을 브라우저에서 열면 됩니다.
